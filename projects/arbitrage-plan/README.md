#### 1.2 发现标准

什么样的产品值得搬运？

✅ 满足以下条件：
- 有实际使用价值
- 国内没有同类产品或体验很差
- 有付费意愿强烈的用户群
- 价格有利润空间（溢价 30%+）
- 不需要复杂部署/技术支持

❌ 不适合：
- 需要绑定国内手机号
- 违反法律法规的产品
- 技术支持需求太高
- 客单价太低（<50元）

### 二、评估阶段

#### 2.1 评估清单

发现产品后，回答以下问题：

1. **这是什么产品？** 核心功能一句话描述
2. **解决什么问题？** 目标用户痛点
3. **国内有竞品吗？** 列出 3-5 个同类产品
4. **我们有什么优势？** 价格/语言/服务/渠道
5. **怎么定价？** 成本 + 合理利润
6. **需要什么交付？** 订阅/安装/培训
7. **能赚多少？** 客单价 × 目标客户数

#### 2.2 定价公式

```
国内售价 = (国外月付价格 × 12) × 1.3 ~ 1.5 + 服务费

例：$9.9/月 → ¥99/月 × 12 = ¥1188/年 × 1.3 = ¥1544/年
```
